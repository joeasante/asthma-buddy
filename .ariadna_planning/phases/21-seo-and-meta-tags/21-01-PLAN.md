<objective>
Add a meta description yield slot to both Rails layouts so that every page can declare its own description without resorting to content_for :head workarounds.

Purpose: All subsequent plans (21-02, 21-03) write `content_for :meta_description` blocks in views. Those blocks only render if the layout contains the corresponding yield. This plan creates the infrastructure both plans depend on.
Output: Two layout files updated; one new line added to each.
</objective>
