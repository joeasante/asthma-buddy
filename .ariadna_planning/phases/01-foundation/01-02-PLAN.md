    ```erb
    <html lang="en">
      <head>
        <title><%= content_for?(:title) ? yield(:title) : "Asthma Buddy" %></title>
        <meta name="viewport" content="width=device-width,initial-scale=1">
        <meta name="apple-mobile-web-app-capable" content="yes">
        <meta name="application-name" content="Asthma Buddy">
        <meta name="mobile-web-app-capable" content="yes">
        <%= csrf_meta_tags %>

        <%#= tag.link rel: "manifest", href: pwa_manifest_path(format: :json) %>

        <link rel="icon" href="/icon.png" type="image/png">
        <link rel="icon" href="/icon.svg" type="image/svg+xml">
        <link rel="apple-touch-icon" href="/icon.png">

        <%= stylesheet_link_tag :app, "data-turbo-track": "reload" %>
        <%= javascript_importmap_tags %>
      </head>

      <body>
        <header role="banner">
          <nav role="navigation" aria-label="Main navigation">
            <%= link_to "Asthma Buddy", root_path, class: "nav-brand" %>
            <%# Authentication links added in Phase 2 %>
          </nav>
        </header>

        <main role="main" id="main-content">
          <% if notice.present? %>
            <p role="status" class="flash flash--notice"><%= notice %></p>
          <% end %>
          <% if alert.present? %>
            <p role="alert" class="flash flash--alert"><%= alert %></p>
          <% end %>

          <%= yield %>
        </main>

        <footer role="contentinfo">
          <p>Asthma Buddy</p>
        </footer>
      </body>
    </html>
    ```

    Key decisions:
    - `lang="en"` added to `<html>` for WCAG 2.2 AA (Phase 9 requirement — cheap to add now)
    - `role="banner"`, `role="navigation"`, `role="main"`, `role="contentinfo"` landmark roles for screen readers
    - `id="main-content"` on main for skip-link target (skip link added in Phase 9)
    - `role="status"` on notice flash, `role="alert"` on alert flash for screen reader announcement
    - Nav comment marks where Phase 2 inserts sign-in/sign-out links
    - PWA manifest comment preserved (Phase 9 enables it)
