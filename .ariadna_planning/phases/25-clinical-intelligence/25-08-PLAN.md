  ```ruby
      add_column :medications, :dose_unit, :string, default: "puffs", null: false
    end
  end
  ```

  Run `bin/rails db:migrate`.

  **2. Add enum and helper to Medication model:**

  In `app/models/medication.rb`, add the enum AFTER the existing `medication_type` enum:

  ```ruby
  enum :dose_unit, { puffs: "puffs", tablets: "tablets", ml: "ml" }, default: :puffs, validate: true
  ```

  Note: This is a string-backed enum (not integer), since the column is a string with default "puffs".

  Add a helper method for display:

  ```ruby
  # Returns human-readable unit label, pluralized based on count.
  # dose_unit_label(1) => "puff", dose_unit_label(2) => "puffs"
  # dose_unit_label(1) => "tablet", dose_unit_label(2) => "tablets"
  # dose_unit_label(1) => "ml", dose_unit_label(2) => "ml"
  def dose_unit_label(count = 2)
    case dose_unit
    when "puffs"
      count == 1 ? "puff" : "puffs"
    when "tablets"
      count == 1 ? "tablet" : "tablets"
    when "ml"
      "ml"
    else
      dose_unit
    end
  end
  ```

  - Invalid `dose_unit` values are rejected by validation
