**Controller (app/controllers/appointment_summaries_controller.rb):**
- Change `@active_courses` query from `user.medications.active_courses.order(:ends_on)` to a query that finds courses overlapping the 30-day period:
  ```ruby
  @period_courses = user.medications.where(course: true)
    .where("starts_on <= ? AND ends_on >= ?", Date.current, period_start)
    .order(:starts_on)
  ```
  This finds courses that started on or before today AND ended on or after the period start date (30 days ago), so any course whose date range overlaps the 30-day window is included.
