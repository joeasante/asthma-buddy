Update `DashboardController#index` to build `@preventer_adherence` — an array of hashes or structs for the view to iterate over.

Add this query AFTER the existing `@low_stock_medications` line:

```ruby
# Today's preventer adherence — only preventers with a doses_per_day schedule
today = Date.current
@preventer_adherence = user.medications
  .where(medication_type: :preventer)
  .includes(:dose_logs)
  .select { |m| m.doses_per_day.present? }
  .map { |m| { medication: m, result: AdherenceCalculator.call(m, today) } }
```

This produces an array of hashes with `:medication` and `:result` keys. The view iterates over this array and renders `_adherence_card` for each entry.

Do NOT use `includes(:dose_logs)` as a caching mechanism for `AdherenceCalculator` — the service queries `dose_logs` directly via `medication.dose_logs.where(...)`. The `includes` here is for the `remaining_doses` and `low_stock?` calls that may also be hit, not for the calculator (which issues its own scoped query). This is acceptable — the adherence section queries are isolated to preventer medications only.

Key constraints:
- Only `preventer` medication type — reliever, combination, and other types are excluded
- Only medications with `doses_per_day.present?` — unscheduled preventers are silently excluded (no display)
- Scoped to `Current.user` via `user.medications` — multi-user isolation maintained

**Update `app/views/dashboard/index.html.erb`:**

Add the adherence section ABOVE the existing `<%# ── Low-stock medications ──` comment (i.e., between the summary stats block and the low-stock section):

```erb
<%# ── Preventer adherence ──────────────────────────────────────── %>
<% if @preventer_adherence.any? %>
  <section class="dash-adherence" aria-label="Preventer adherence today">
    <div class="dash-section-header">
      <h2>Preventers Today</h2>
      <%= link_to "View history", adherence_path, class: "dash-section-link" %>
    </div>
    <ul class="dash-adherence-list">
      <% @preventer_adherence.each do |entry| %>
        <%= render "dashboard/adherence_card",
              medication: entry[:medication],
              result: entry[:result] %>
      <% end %>
    </ul>
  </section>
<% end %>
```
